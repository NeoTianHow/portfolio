## Projects

### PSA CodeOptics - Capstone Project

**Associated with Singapore Institute of Technology**  
November 2024 - March 2025

Developed CodeOptics as a capstone project for internal use by my department at PSA. CodeOptics is a platform designed to improve repository visibility and development context across hundreds of GitLab repositories. The system provides an interactive dashboard that surfaces real-time branch activity, including merged, unmerged, active, and stale branches, helping developers quickly identify outdated or unmerged work.

Integrated Qwen-2.5-Coder, an LLM, to generate natural language summaries of branch context and key code changes using preprocessed code diffs and commit messages. This helped developers understand the purpose and scope of unfamiliar branches without manually reviewing code line by line.

Implemented AI-based code reviews on GitLab merge requests, triggered via webhook, with comments generated based on PSA's internal coding standards. To improve the system over time, I built a feedback mechanism that captures downvoted review comments directly from the GitLab UI via webhook, stores them as vector embeddings in Chroma, and uses them for continuous refinement of the LLM output.

Used JasperReports to generate structured PDF summaries of branch activity for documentation and reporting purposes.

**Technologies:** Chroma, Ollama, Artificial Intelligence (AI), Object-Oriented Programming (OOP), Node.js, H2 Database, Spring Boot, Spring AI, JavaScript, SQL, AngularJS
